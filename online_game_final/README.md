# dee-first
Project 8. 
А. Python. Analysis of the effectiveness of different user acquisition advertising channels and visualized ranking of channels based on their effectiveness. Development of a dashboard and presentation.

Заказчик: «Космические братья»

Описание:

Проведена рекламная кампания с различными источниками привлечения пользователей в онлайн игру. В процессе игры игроки добиваются победы 2 путями: военная победа(PvP) или завершение проекта(PvE). Для любого из путей необходимо строительство зданий. Реклама показывается при строительстве каждого здания.

B. A/B test

С. SQL. Analysis of the database of a book reading application.
